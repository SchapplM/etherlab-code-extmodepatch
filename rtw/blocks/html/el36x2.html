Beckhoff EL36x2 24-bit analog input terminal

The EL36x2 family of input slaves are high resolution wide range analog input terminals. Supported slaves are:
  • EL3602: +-10V, +-5V, +-2.5V +-1.25V voltage input
  • EL3602-0002: +-200mV voltage input
  • EL3602-0010: +-75mV voltage input
  • EL3602-0020: EL3602 with calibration certificate
  • EL3612: 0..20mA current input
  • EL3612-0020: EL3612 with calibration certificate

Apart from the common slave options, the following options configure the slave:

Vector output

The channels are presented as a vector or scalars for further processing.

Status port

Output status information (PDO Entry #x6000:7)

1 Ch. only

Only use first input channel (reduces EtherCAT frame size, increases conversion speed)

CoE/Filter

Choose a filter frequency. Filter is always active, see documentation.

CoE/Range

Select voltage range (EL3602 only).
