Beckhoff EL3255 Potentiometer input terminal

The EL3255 is a 5-channel input slave allowing direct connection of potentiometers. Using distributed clocks, highly synchronized measurement of all channels are possible. The block presents a ratiometric output port corresponding to the number of channels selected.

Apart from the common slave options, the following options configure the slave:

Active Channels

Choose the number of channels. This improves the performance as well as preventing a red led status from reporting a disconnected state.

Vector Output

Collect all outputs to a single vector.

CoE Configuration

Here the slave is configured according to the documentation.
