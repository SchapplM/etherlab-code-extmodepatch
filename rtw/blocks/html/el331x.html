Beckhoff EL331x Thermocouple input terminal

The EL331x series input slave is a temperature measurement slave that allows a wide range of thermocouples as a sensor. Information for individual channel configuration is available in the documentation. All channels are mapped.

Apart from the common slave options, the following options configure the slave:

Vector Output

Collect all outputs (and inputs) to a single vector.

Status Output

Error status of channel (disconnected, overrange, underrange). This is only available when vector output is chosen.

CoE Configuration

In tabs Ch.1 .. Ch.8, the individual channels are configured.

The temperature output is scaled correctly according to SDO #x80n0:02.

If external cold junction compensation (SDO #x80n0:0C) is chosen for any channel, an input port becomes available where the external temperature (in degrees) can be transmitted to the slave.
