Beckhoff EK1914 coupler with standard and safe IO

Please read the appropriate documentation for this terminal which will help in configuring it. In the Main tab, configure common slave options as well as:
  • Standard IO will add a port for standard digital IO
  • Safe IO will add ports where the safe outputs can be modified and the status of the safe inputs are reflected.
The upper input and output ports must be connected to the EL6900 Safety Master.
