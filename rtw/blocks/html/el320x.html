Beckhoff EL320x 1-, 2-, 4-Channel Input Terminal Pt100 (RTD)

The EL320x series of analog input terminals measure resistance sensors and directly convert it to a temperature. A wide range of characteristic sensor curves are supported. 2-, 3- and 4-wire connection technologies can be used.

The temperature value is directly available as a double value on the first output port, correctly scaled.

Apart from the common slave options, the following options can configure the slave.

Consult the slave documentation for this slave.

RTD-Element

The RTD-Element type is a number that is written in CoE object #x80n0:19 that represents the resistance sensor.

This is either a scalar or a vector that represents the type of RTD element that is connected. A scalar will be replicated to all channels. A vector must have tha same number of elements as the slave has channels

Connection Technology

The connection technology is a number that is written in CoE object #x80n0:1A and represents how the sensor is connected.

This is either a scalar or a vector. A scalar will be replicated to all channels. A vector must have tha same number of elements as the slave has channels

Here is an extract of the Beckhoff documentation of the connection technology:

TypeDescription
0: 2-wire
1: 3-wire
2: 4-wire
3: not connected

Filter

Choose an appropriate notch filter frequency. The filter applies for all channels.

Status Port

Available only when Vector Output is selected, a second port indicates the channel's status.
