Murr Impact67 Digital IO Slave

This block adds support for Murr Digital IO field box. The following Terminals are supported:
DI8DO8 8 Input 8 Output Ports
DO16 16 Output Ports
DO8 8 Output Ports
DI16 16 Input Ports

Common Slave Options

  • Master
  • Slave Index
  • Device Model
  • Vector Outputs
  • Sample Time

Port arrangement

Vector IO is enabled:
Input ports: If the model has output terminals (DI8DO8, DO16, DO8), one input port will appear.

Output ports:

  • if the model has input terminals (DI8DO8, DI16), the first port is the logical status of the input pin.
  • Next follows a model status port. Depending on the model, this port will have between 2 and 4 status bits.
    DI8DO8 Undervoltage US
    Undervoltage UA
    Sensor Shortcut (global)
    Actuator Shortcut (global)
    DO16/DO8 Undervoltage US
    Undervoltage UA
    Actuator Shortcut (global)
    DI16 Undervoltage US
    Sensor Shortcut (global)
  • Next follows a output terminal status port if the terminal has outputs.
If Vector IO is not enabled, the input and output ports will be split up in the order listed above
