Beckhoff EP43xx 2Ch Analog Input/Output Slave

This block implements the Beckhoff's EtherCAT EP43xx series of Analog Input and Output fieldbus module.

The module has 2 configurable analog input ports and 2 configurable analog output ports. Choose the correct configuration using the provided selection lists.

The analog input ports also have a Status Output that can be activated. This presents binary PDO Entry #x60n0:7 on the lower half of the block's outputs. According to the documentation, if the input range of an analog input is exceeded, this value goes high. Choosing the status does come at a small cost of requiring a 4-byte PDO per channel instead of 2-bytes only.
