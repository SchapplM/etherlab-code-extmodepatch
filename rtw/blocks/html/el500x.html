Beckhoff EL500x SSI Interface

This block implements the Beckhoff's EtherCAT series of SSI Sensor Interface (EL5001, EL5002) and Monitor Terminal (EL5001-001x)

Sensor Interface

 Configure the terminals EL5001 and EL5002 in the tabs
  • Main Note that for DC, do not use DC Latch active. This is reserved for EL5001-001x
  • Chan1 See the Beckhoff documentation for a description of the various fields
  • Chan2 Only for EL5002
The following block outputs appear:
  1. SSI code (PDO #x6000:17)
  2. Data Error (PDO #x6000:1)
  3. Frame Error (PDO #x6000:2)
  4. Power Failure (PDO #x6000:3) - check option Enable Power Failure bit in ChanX tab
For EL5002, every port is a vector with 2 elements.

Monitor Terminal

 Configure the EL5001-001x terminal in the tabs
  • Main Note that for DC, only the options
    • FreeRun/SM-Synchron
    • DC Latch active
    • DC Customized with #x120 for AssignActivate
    should be used
  • Chan1 Note that Baudrate and Min Inhibit Time are ignored.
  • EL5001-001x Note that to be able to use Timestamp, choose the correct Distributed Clocks mode other than FreeRun/SM-Synchron in the Main tab.
The block outputs are the same as for Sensor Interface, except that if Timestamp is chosen, PDO #x1a02:6000:18 is inserted at Port 2.
